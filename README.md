# Shepherd Pocket Jump

一个 Godot 4.x 的像素风跳跃收集小游戏原型。主题是放羊：牧羊人沿着山坡平台向上跳，把散落在山上的羊一只只收入口袋，收满后到达山顶。

## 运行

1. 用 Godot 4.x 打开本文件夹。
2. 运行 `res://scenes/main.tscn`，或直接点击运行项目。

## 操作

- `A/D` 或方向键左右移动
- `Space/W/Up` 跳跃
- `R` 重新开始

## 设计方向

- 视觉：低饱和绿色为主，背景山体和草地平台保持像素块风格，牧羊人和小羊使用 `素材/processed` 中的透明抠图 sprite。
- 玩法：轻量平台跳跃，带土狼时间和跳跃缓冲，让连续上山更顺手。
- 开局：每次开始会先自动向右运镜预览到山顶终点，再回到起点恢复玩家操作。
- 目标：收集随机出现在山坡平台上的羊，口袋计数满后抵达山顶旗帜。
- 牧羊人：初始为站立；向左/右移动时进入 4 帧赶路动画；无操作 12 秒后休息，无操作 20 秒后睡觉。站立、休息和睡觉时会有轻微上下律动。
- 小羊：只保留睡觉、吃草、走路和奔跑四类状态；睡觉和吃草会有轻微起伏，走路和奔跑会左右移动。
- 生成规则：每局随机分布 7 只小羊，但同一处羊群最多 3 只，并避开牧羊人出生点附近；同平台小羊初始位置会保持间距，走路和奔跑的小羊允许互相穿过和遮挡。
- 扩展点：可以继续加入移动平台、风、牧羊犬辅助、羊的不同颜色或计时评价。
